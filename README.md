# Deep Work — Premium Study Timer

A modern, visually stunning, distraction-free study timer designed for deep focus and flow states.

## ✨ Features
- **Zero-Drift Accuracy:** Calculates time directly against your device's system clock. Perfect accuracy even if you switch tabs or the browser sleeps.
- **Day & Night Auras:** Beautiful, soft-glowing radial backgrounds. Warm amber/orange for focused study, cool violet/indigo for breaks.
- **Dual Panel Break Mode:** Seamless transition to a split view during breaks, tracking both elapsed study time and the current break length.
- **Persistent Sessions:** State natively auto-saves to your browser. Accidental reloads will never reset your timer!
- **Fluid Micro-Animations:** Satisfying button feedback, rolling character digits on the timer, and physics-based modals powered by Framer Motion.
- **Session Summaries:** Comprehensive end-of-session breakdown of total focus time vs break time, including an earned qualitative Focus Rating.

## 🛠️ Tech Stack
- Next.js (App Router)
- Tailwind CSS
- React Hooks + LocalStorage
- Framer Motion

## 🚀 Getting Started
```bash
npm install
npm run dev
```
